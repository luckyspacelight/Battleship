Here's a class hierarchy consisting of four classes.

class A {
    
    public void print() {
        System.out.println("class-A");
    }
}

class B extends A { }

class C extends B {
    
    public void print() {
        System.out.println("class-C");
    }
}

class D extends C {

    public void print() {
        System.out.println("class-D");
    }   
}

What does the code below output?

B instance = new C();
instance.print();
